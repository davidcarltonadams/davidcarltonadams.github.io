tools

browser-based instruments and utilities for performers, composers, and anyone who thinks in sound.

  • clock

    Performance Clock

    Synchronized clock for ensemble performance. Define sections with text or image content — rehearsal letters, score pages, improvisation prompts. One conductor starts; everyone connected sees the same clock and the same score. Works over any internet connection. Performers scan a QR code to join. Per-section passwords for staged reveals.

    ensemble real-time improvisation conductor score display
  • score

    Performance Instructions

    Thirty graphic score prompts drawn from the vocabulary of an extended practice: microtonality, emergence, duration, ensemble relations, voice, silence. Random on load. Each instruction has its own URL — link to one directly with ?n=14. Changes every five minutes on its own.

    score improvisation text score indeterminacy
  • notation

    Generated Score Instructions

    Procedurally generated performance instructions built from the structural patterns and vocabulary of the practice. Every instruction is unique. Shareable via URL seed — if you find one you like, the URL will regenerate it exactly.

    score generative improvisation
  • drone

    Drone Machine

    Pure tone drone with just intonation tuning, variable beating, and harmonic partial toggles. Choose a root from C through B, select an octave, dial in a beat rate. Add upper partials (2nd through 7th) to build a harmonic field. Useful for ear training, tuning, and meditative listening.

    tuning just intonation ear training drone
  • spectrogram

    Real-Time Spectrum Analyzer

    Microphone input, FFT analysis on a log-frequency scale with overlaid just intonation partial markers. Choose a root pitch; markers show where partials 1–16 fall relative to your signal. Useful for microtonal ear training, tuning investigations, and understanding what your instrument is actually doing. Listens only — no audio output.

    analysis microtonality ear training mic input
